# tpt-augur-cli

Command-line interface for the [Augur](https://github.com/tpt-solutions/tpt-augur)
probabilistic programming language: `run`, `check`, `fmt`, and `repl`
subcommands, installed as the `augur` binary.

```sh
cargo install tpt-augur-cli
augur check model.augur
```

See the [main repository](https://github.com/tpt-solutions/tpt-augur) for the
language overview, worked examples, and docs.
