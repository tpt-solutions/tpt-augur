# tpt-augur-lsp

Language server for the [Augur](https://github.com/tpt-solutions/tpt-augur)
probabilistic programming language: diagnostics, formatting, and completion
over the standard Language Server Protocol.

Editor integrations (VS Code, Neovim) live under `editors/` in the
[main repository](https://github.com/tpt-solutions/tpt-augur).
